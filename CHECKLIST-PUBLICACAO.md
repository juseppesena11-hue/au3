# Checklist de publicação

## Conteúdo e legal

- [ ] Nome jurídico final confirmado.
- [ ] NIF confirmado.
- [ ] Regime de IVA confirmado.
- [ ] Condições reais de garantia/assistência confirmadas.
- [ ] Preços da calculadora validados.
- [ ] Páginas legais revistas por responsável adequado.
- [ ] Política real de conservação de dados e anexos definida.
- [ ] Logótipo definitivo substituído, se existir.
- [ ] Fotografias publicadas revistas para privacidade.
- [ ] Municípios efetivamente atendidos confirmados.
- [ ] Testemunhos reais adicionados apenas com autorização.

## Técnica

- [ ] `npm install`
- [ ] `npm run lint`
- [ ] `npm run build`
- [ ] Variáveis SMTP configuradas na Vercel.
- [ ] Formulário testado sem anexos.
- [ ] Formulário testado com até 3 imagens.
- [ ] Fallback WhatsApp testado.
- [ ] WhatsApp e telefone testados.
- [ ] Mobile e desktop revistos.
- [ ] `robots.txt` verificado em produção e preview.
- [ ] `sitemap.xml` contém apenas conteúdos publicados.
- [ ] Canonicals apontam para cada página correta.
- [ ] Páginas draft devolvem 404 e não entram no sitemap.
- [ ] Risco residual do Next.js 14 revisto; atualização para versão suportada planeada antes do lançamento, se necessário.

## SEO e medição

- [ ] Domínio `aureon.pt` ligado e HTTPS ativo.
- [ ] Search Console validada.
- [ ] Sitemap enviado.
- [ ] Open Graph testado.
- [ ] Analytics só ativado depois de decisão sobre consentimento e atualização da política de cookies.

## Operação

- [ ] Processo de resposta a pedidos definido.
- [ ] Responsável por atualizar preços e conteúdos identificado.
- [ ] Rotina para publicar artigos e projetos reais definida.
- [ ] Arquivo fotográfico futuro organizado por obra, antes/durante/depois e autorização.
